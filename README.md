# 🛍️ Fashion Store – E-Commerce Web Application

A full-stack Java web application that enables users to browse fashion products, manage their shopping cart, and place orders. The application follows the MVC architecture and is built using Java, Servlets, JSP, JDBC, and MySQL.

---

## 📌 Features

- 👤 User Registration & Login
- 🛒 Add to Cart & Cart Management
- 👕 Product Listing with Images
- 🔍 Search Products
- 🏷️ Category-wise Product Filtering
- 💳 Checkout & Order Placement
- 📦 Order Confirmation
- 🗄️ MySQL Database Integration
- 🏗️ MVC Architecture

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Sanjushree-T/Fashion-Store-Ecommerce-Web-Application.git
```

### 2. Import into Eclipse

- File → Import
- Existing Projects into Workspace

### 3. Configure Database

- Install MySQL
- Create the database
- Import the SQL script
- Update database credentials in the project

### 4. Configure Apache Tomcat

- Install Apache Tomcat 10+
- Add Tomcat Server in Eclipse
- Deploy the project

### 5. Run

```
http://localhost:8080/FashionStore
```
